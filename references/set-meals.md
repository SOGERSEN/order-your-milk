## How to Use Set Meals

When the user's requirement is too vague to choose a specific option:

```
Agent: "I have 16 ready-made set meals in three tiers by difficulty — see if any fits:"

🟢 Beginner (simple & fast):
  🅰 Personal Showcase — 10min, resume/homepage
  🅱 Small Utility — 15min, calculator/converter
  🅲 Admin Panel — 30min, data tables
  🅳 Data Dashboard — 20min, chart visualization
  🅴 Landing Page — 20min, product promo

🟡 Intermediate (some tech skills needed):
  🅵 Full-Stack App — 60min, complete frontend+backend app
  🅶 CLI Tool — 15min, command-line tool
  🅷 API Service — 30min, backend API
  🅸 Data Analysis — 20min, Pandas + charts
  🅹 Chatbot — 25min, AI chat interface
  🅺 E-Commerce Frontend — 40min, products + cart
  🅻 Learning Platform — 45min, courses + quizzes

🔴 Professional (domain-specific):
  🅼 Docs/Wiki — 30min, knowledge base docs site
  🅽 Mobile H5 — 25min, mobile pages
  🅾 Game/Interactive — 30min, HTML5 mini-game
  🅿 Automation Workflow — 20min, scripts + scheduled tasks

Which one catches your eye? Or tell me your specific needs and I'll customize it.
```

After the user picks a meal, go directly to Step 3 (execution) — skip the detailed ordering.
If the user wants modifications to a meal, make adjustments on top of the meal preset.
